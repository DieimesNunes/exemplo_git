## Criar um novo repositório - Comandos GIT

No terminal (Só fazer se for novo projeto, para projeto que já existe, não fazer).
git init
git add .
git commit -a -m “mensagem”
(copiar do github) git Branch -M main
(copiar do github ) git remote add origin https://github.com/DieimesNunes/exemplo_git.git (Pegar o seu endereço)
git push -u origin main
